# 盖亚爷爷桌宠 (Gaia Grandpa Desktop Pet)

一个运行在桌面上的赛尔号战神联盟盖亚爷爷桌面宠物。

## 功能

- 实时显示在桌面上的 GIF 动画
- 支持 AI 对话（通过 codebuddy / GLM-5.1 / OpenRouter API）
- 深色主题对话框
- 右键菜单（隐藏/关闭/开机自启）
- 多行输入框（Ctrl+Enter 发送）

## 运行

```bash
python desktop_pet.py
```

或者双击运行 `desktop_pet.pyw`（无窗口版本）。

## 技术栈

- Python 3
- tkinter（GUI）
- codebuddy CLI（AI 对话）
- OpenRouter API（minimax 免费模型）
